# VibeCheck

VibeCheck is an AI-powered interview practice platform designed to help students improve communication presence in a pressure-free environment.

## Overview
The platform allows users to respond to interview questions while observable communication signals such as eye focus, head movement, and speech activity are analyzed. These signals are converted into structured, supportive coaching feedback rather than scores or judgments.

## Development Approach
This project was built using a *vibecoding workflow*, where AI-assisted coding tools were used to accelerate development and iteration. All system design decisions, signal analysis logic, feedback constraints, and user experience flows were explicitly designed and controlled by the developer.

AI was used as a development accelerator, not as an autonomous decision-maker.

## Technologies Used
- HTML, CSS, JavaScript
- MediaPipe Face Mesh (facial landmark detection)
- Gemini API (AI-generated coaching feedback)

## Responsible AI
- AI acts as a coaching assistant, not an evaluator
- No scoring, ranking, or hiring decisions
- Feedback depends on signal availability and confidence
- Privacy-first, user-focused design
